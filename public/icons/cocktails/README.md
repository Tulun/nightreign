# Cocktail icons

Optional icons for Recluse's Magic Cocktails. Drop a square PNG named by the
cocktail id and it loads automatically (the card derives the path):

- e.g. `molotov.png`, `tracking-wisp.png`, `infinite-mana.png`

Ids are in `src/data/cocktails.ts` (lowercase, hyphenated). Until a file exists,
a placeholder flask glyph is shown.
