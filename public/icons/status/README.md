# Status-effect icons

Drop square PNGs here (≈32×32 looks best). The app references these exact
filenames — until a file exists, chips fall back to text-only automatically:

| File          | Status        | Notes                          |
|---------------|---------------|--------------------------------|
| `poison.png`  | Poison        | green                          |
| `rot.png`     | Scarlet Rot   | red/orange                     |
| `bleed.png`   | Blood Loss    | red splatter                   |
| `frost.png`   | Frostbite     | blue                           |
| `sleep.png`   | Sleep         |                                |
| `madness.png` | Madness       | gold eye                       |

Element/affinity icons (Magic, Fire, Lightning, Holy) live in
`public/icons/elements/` and are already wired in.
